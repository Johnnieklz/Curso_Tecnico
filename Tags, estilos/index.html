JavaScript

É imposivel falar sobre linguagens de programação e não mencionar a JavaScript, a mais famosa e utilizada em todo o mundo. Esse é um exelente recurso para desenvolver soluções front-end e back-end.

Python

Python é uma linguagem de programação poderosa e fácil de aprender. Ela possui uma sintaxe simples e por isso é muito utilizada pelas mais diversas áreas da computação para diversos programas.

PHP

PHP é uma linguagem de programação de codigo aberto, criada para o desenvolvimento web. Com ela podemos escrever pequenos scripts de forma procedual, utilizar orientação a objetos ou ainda ambas.
